# MVC CRUD Application with HikariCP

This is a simple CRUD (Create, Read, Update, Delete) application implemented in Java using the MVC architecture and HikariCP for connection pooling.

## Prerequisites

- Java Development Kit (JDK) version 8 or higher
- Apache Maven for dependency management (optional, you can also use Gradle or other build tools)
- MySQL or any other relational database management system

## Usage

- You can perform CRUD operations on the `items` table using the web interface.
- Create: Click on the "Add Item" button and fill in the details.
- Read: View all items listed on the homepage.
- Update: Click on the "Edit" button next to an item to modify its details.
- Delete: Click on the "Delete" button next to an item to remove it from the list.

## Dependencies

- Spring Boot: Framework for creating stand-alone, production-grade Spring-based applications.
- HikariCP: High-performance JDBC connection pool.
- Thymeleaf: Modern server-side Java template engine for web and standalone environments.
